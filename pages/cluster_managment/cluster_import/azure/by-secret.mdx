#### Import Cluster by Secret:

To import a cluster using a secret, click on the Secret option.

![Import-Cluster](/import_cluster/import-cluster3a.png)<br />

* A window will appear displaying the available cloud providers (Amazon, Azure, DigitalOcean, Linode, Google). Select the <b> Azure </b> to import your cluster.

![Import-Cluster](/import_cluster/import-cluster2.png)

* The user must first select <b>Secret</b> from the Secret dropdown, followed by choosing the appropriate <b>Region </b>from the Region dropdown. After making the selections, click the Next button to proceed

<br />
![Import-Cluster](/import_cluster/azure-secret/azure-import-secret0.png)<br />

* The next step is Cluster Configuration, where the cluster status will change from Creating to Running. Once the status is updated, click the Next button to proceed to the subsequent steps.

<br />
![Import-Cluster](/import_cluster/azure-secret/azure-import-secret1.png)<br />

* In the final step, click the Submit button to complete the process and finalize the cluster import.
<br />
![Import-Cluster](/import_cluster/azure-secret/azure-import-secret2.png)<br />
