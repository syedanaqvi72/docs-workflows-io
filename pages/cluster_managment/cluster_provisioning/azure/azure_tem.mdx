#### Azure Cluster Provisiong by Template

- Navigate to the <b>"Cluster" </b>tab on the dashboard.
- Within the "Cluster" section, you will find a button labeled <b>"All Cluster".</b> Click on this button to initiate the cluster creation process
- In the upper-right corner, click the <b>"Create"</b> button.

![azure](/cluster_provisioning/azure/azurebasic1.jpg)

- Subsequently, a pop-up window will appear, presenting a selection of various cloud providers. Please proceed by selecting cloud provider, i.e.<b> Azure.</b>

![azure-basic](/cluster_provisioning/azure/azurebasic2.jpg)

- After selecting Azure, you will encounter a new pop-up window where you will be presented with  distinct options: <b>"Basic" , and "Templates."</b>
* Select Template.

![azure](/cluster_provisioning/azure/template/azuretem0.png)<br />

* Upon initiating the cluster provisioning process using a template, a window appears displaying the steps.<br />
 * The first step is <b>Select Template" </b>From the available options, choose the desired Azure template and click the <b>"Next"</b> button to proceed.

![azure](/cluster_provisioning/azure/template/azuretem1.png)<br />
* In the second step, <b>Update Configuration</b>, the user can modify the configuration in the editor if needed, and then click the <b>Deploy Configuration</b> button to proceed.
![azure](/cluster_provisioning/azure/template/azuretem2.png)<br />
*  A confirmation message will then appear, indicating that the Azure template cluster creation request has been submitted.
![azure](/cluster_provisioning/azure/template/azuretem3a.png)
<br />
* The system then redirects to the<b>Cluster Details</b>, where the cluster status appears as<b>"Provisioning."</b>
<br />
![azure](/cluster_provisioning/azure/template/azuretem3.png)
<br />
* After a brief period, the status will transition from<b>Provisioning</b> to <b>Running </b>and the deployed template will be listed in the <b> Cluster</b> section.
<br />
![azure](/cluster_provisioning/azure/template/azuretem4.png)
<br />
