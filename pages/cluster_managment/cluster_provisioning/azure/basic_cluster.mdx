#### Basic Cluster Configuration Guide

-  Navigate to the <b>"Cluster" </b>tab on the dashboard.
- Within the "Cluster" section, you will find a button labeled <b>"All Cluster".</b> Click on this button to initiate the cluster creation process
- In the upper-right corner, click the <b>"Create"</b> button.

![azure-basic](/cluster_provisioning/azure/azurebasic1.jpg)

- Subsequently, a pop-up window will appear, presenting a selection of various cloud providers. Please proceed by selecting cloud provider, i.e.<b> Azure.</b>

![azure-basic](/cluster_provisioning/azure/azurebasic2.jpg)

- After selecting Azure, you will encounter a new pop-up window where you will be presented with distinct options: <b>"Basic" and "Templates."</b>

![azure-basic](/cluster_provisioning/azure/template/azuretem0.png)

## Basic Kubernetes Cluster

Creating a Basic Azure Cluster involves users meticulously filling out a comprehensive form in a pop-up window. This includes essential details such as the cluster name, secret, chosen region, assigned resource group, number of nodes, and machine specifications. Upon carefully entering this crucial data, users can initiate the establishment of the Azure cluster by pressing the <b>'Submit' </b>button, marking the formal beginning of the process.

![azure-basic](/cluster_provisioning/azure/azurebasic4.jpg)
