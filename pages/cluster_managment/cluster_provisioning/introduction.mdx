### Create Kubernetes Cluster

###### Creating a Kubernetes cluster involves:

- Select a cloud provider or infrastructure platform where you want to create your Kubernetes cluster. Common choices include AWS (Amazon Web Services), GCP (Google Cloud Platform), Azure, DigitalOcean, or Linode.

![select_cluster](/cluster_provisioning/image1.png)

- Create your first kubernetes cluster by following below guids

| Cloud Provider                                 | Create                                                                         |
| ---------------------------------------------- | ------------------------------------------------------------------------------ |
| AWS EKS                                        | [EKS Cluster](/cluster_managment/cluster_provisioning/amazon)                  |
| Google Kubernetes Engine (GKE)                 | [GKE Cluster](/cluster_managment/cluster_provisioning/google)                  |
| Azure Kubernetes Service (AKS)                 | [AKS Cluster](/cluster_managment/cluster_provisioning/azure).                  |
| DigitalOcean Kubernetes Service (DigitalOcean) | [DigitalOcean Cluster](/cluster_managment/cluster_provisioning/digital_ocean). |
| Linode Kubernetes Service (LKE)                | [Linode Cluster](/cluster_managment/cluster_provisioning/linode).              |
