#### Template Tree 
* To add a package in a template, click the desired template in which you want to include the package  or application .
<br  />
![template](/template/tree/tree1.png) <br />
###### Add Packages in the Template 
* To add an package , click on <b>"+."</b>
 <br />
![template](/template/tree/tree2.png) <br />
* A window will appear with three steps.
*  The first step is to select the <b>repository.</b> Choose a repository from the list, and a double checkmark will appear on the selected repo, or click <b>Add Package Repo </b>to add a new repository.
* After Selecting . Click on <b> Next Step </b>
<br />
 ![template](/template/tree/managetree1.png) <br />
 * The second step is to <b>select a package</b> from the list. A double tick will appear on the selected package. 
 * Then, click <b>'Next Step' </b>to proceed
 <br />
 ![template](/template/tree/managetree2.png) <br />
* The final step is to configure the package. Select the version and click the<b> 'Generate Manifest'</b> button.
 <br />
 ![template](/template/tree/managetree3.png) <br />
* A confirmation pop-up window appears, displaying the generated manifest.
* Click on <b>"Add Cluster"</b> button.
 <br />
 ![template](/template/tree/managetree4.png) <br />
* Pop-up windows for packages and the manifest added will appear.
 <br />
 ![template](/template/tree/managetree5.png) <br />
* The added cluster and manifest will also be visible in the template tree.
 <br />
 ![template](/template/tree/managetree6.png) <br />
###### Add Application in the Template 
* To add an application, click on <b>"Add Types."</b>
 <br />
![template](/template/tree/tree2.png) <br />
* A pop-up window will appear. Select <b>"Application" </b>as the type.
 <br />
![template](/template/tree/tree3.png)<br />
* Click on <b>"Add Type"</b> to confirm your selection and proceed.
<br />
![template](/template/tree/tree4.png)
<br />
* A confirmation window for the added packages will appear.
<br />
![template](/template/tree/tree5.png)
<br />
* The <b>application</b> will now be visible in the Template Tree, reflecting the updated structure.
<br />
![template](/template/tree/tree6.png) 
<br />
* Click the Add <b>(+) </b>button next to the application to configure settings.
<br />
![template](/template/tree/tree7.png)
<br />
* A window will appear with two steps.
* First, select the desired application from the list. A double checkmark will appear on the application you select.
* Click on <b> Next Step</b>
<br />
![template](/template/tree/tree8.png)
<br />
* The second step is to configure the application.
* A form will appear with the application name, version, and namespace automatically filled.
* Once the details are verified, click <b>Add Application</b> to complete the process.
<br />
![template](/template/tree/tree9.png)
<br />
* A confirmation pop-up window for the application will appear.
<br />
![template](/template/tree/tree10.png)
<br />
* The name of the application will appear in the Template Tree, indicating that the application has been successfully added.
<br />
![template](/template/tree/tree11.png)
<br />
* If you want to add a manifest to the application, click the <b>Add (+)</b> button.
<br />
![template](/template/tree/tree12.png)
<br />
* A form will appear asking for the manifest name and namespace. Add the manifest in the editor, then click the <b>Update Manifest</b> button to save the changes.
<br />
![template](/template/tree/tree13.png)
<br />
* A confirmation message will appear, confirming that the manifest has been successfully added.
<br />
![template](/template/tree/tree14.png)
<br />
* The name of the manifest will appear in the Template Tree, indicating that the manifest has been successfully added to the application.
<br />
![template](/template/tree/tree15.png)
