#### Import Template

In addition to creating new templates, Gripo offers the flexibility to import templates. This feature enables users to quickly integrate pre-configured templates from external sources, ensuring seamless cluster provisioning across various cloud platforms, including Amazon, Azure, DigitalOcean, Google, and Linode.

##### Import To Gripo:

- To import a template in Gripo, click on the <b>"Import from File"</b> option in the template section, allowing you to upload an existing template for use in cluster provisioning.

  <br />
  ![template](/template/tem/tem2.jpg)
  <br />
- After clicking on "Import from File" in Gripo, a window will open offering two options. 
* You can either click the  <b>"Choose File"</b> button to select your JSON file from your local machine, or you can <b>drag and drop</b> the JSON file directly .

  <br />
  ![template](/template/import/importtem0.png)
  <br />
  <b>Drag & Drop:</b> 
  To upload using drag and drop, just drag the template .json file from your system and drop it into the upload area. The file will be uploaded instantly.<br />
<b>Select Json File :</b>
Click Select File, then choose the template .json file saved on your computer to begin the import process.
 <br />
  ![template](/template/import/importtem01.png)
  <br />
- In the import window, enter a name for your template in the <b>"Template Name"</b> field 
* Provide a description in the <b>"Description"</b> field. 
* Afterward, click on the <b>"Next Step" </b>button. 
* If you select the wrong JSON file, you can click on the <b>"Select Another Template" </b>option to choose a different file.
