##### Create Amazon Template

- To create a new Amazon template, click on the <b>"Create New Template"</b> button in the <b>Templates section </b>.

  <br />
  ![template](/template/tem/tem2.jpg)
  <br />

- When you click on "Create New Template" for Amazon, a form will appear. Fill in the required fields, including the <b>name, version, and description</b> of your template. In the <b>"Provider"</b> field, choose "Amazon."

  <br />
  ![template](/template/createtemplate/createtem1.png)
  <br />

- In the <b>"Template Type"</b> field, select "Managed Cluster,"
  {" "}
  <br />
  ![template](/template/createtemplate/createtem2.png)
  <br />
- After completing the form, click on "Create Template" to save it.
  {" "}
  <br />
  ![template](/template/createtemplate/amazontem1.png)
  <br />
##### In the template creation process, follow these three steps:<br />
  * <b>Choose Your Repository:</b> Select your repository from the list. A double tick
  will appear next to your selected repository.
  *  Click "Next Step."
  <br />
  ![template](/template/createtemplate/createtem4.png)
  <br />
  * <b>Select the Package:</b> Choose the package from the list. A double tick
  will appear next to your selected package. 
  * Click "Next Step."
  <br />
  ![template](/template/createtemplate/packageamazon.jpg)
  <br />
  <b>Configure Package:</b>

A form will appear with the following fields:

- Name: Enter a name for your template.
- Package Version: Select the version of the package.
- Provider: Amazon.
- Secret: Input any necessary secrets for the template.
- Region: Select the AWS region for deployment.
- Kubernetes Version: Choose the Kubernetes version for your cluster.
- Service Role: Define the service role for the cluster.
- Node Role: Set the role for the worker nodes.
- Node Size: Select the EC2 instance size for your nodes.
- Node Group Name: Define a name for your node group.
- Desired Capacity: Specify the desired number of nodes in your cluster.
- VPC: Choose the Virtual Private Cloud (VPC) for your cluster.
- Subnets: Select the subnets for the nodes.
- Security Group: Choose the security group for your cluster.
- After filling out the necessary fields, click on Generate Packages on the editor.
  {" "}
  <br />
  ![template](/template/createtemplate/amazontem2.png)
  <br />

 * Click on Generate Manifest to complete the template creation.
