#### Managed Cluster-Template

Gripo's ManagedCluster templates allow users to define and reuse standardized cluster configurations across multiple cloud providers for streamlined provisioning.


In Gripo, users can create Managed-cluster templates for clusters on various cloud platforms, including <b>Amazon, Azure, DigitalOcean, Google, and Linode.</b>

| Cloud Provider                                 | Templates                                                                         |
| ---------------------------------------------- | --------------------------------------------------------------------------------- |
| AWS EKS                                        | [EKS Template](/template/create_template/managed_cluster/amazon)                |
| Google Kubernetes Engine (GKE)                 | [GKE Template](/template/create_template/managed_cluster/google)                |
| Azure Kubernetes Service (AKS)                 | [AKS Template](/template/create_template/managed_cluster/azure)                 |
| DigitalOcean Kubernetes Service (DigitalOcean) | [DigitalOcean Template](/template/create_template/managed_cluster/digitalocean) |
| Linode Kubernetes Service (LKE)                | [Linode Template](/template/create_template/managed_cluster/linode)             |
