# Install Gripo with Docker 

You can install and try Gripo on any machine running docker and docker-compose.
<br />

  <TabList>
    <Tab>MacOS</Tab>

  </TabList>

  <TabPanel>
    ### Prerequisite

It is essential to install the following applications on your macOS, Windows or Linux machine before proceeding

* Install [Docker and Docker Compose](https://docs.docker.com/desktop/).
* Install [Github Desktop ](https://desktop.github.com/download/).

### Clone repo

Open the command prompt and run the following command to clone the repository.

```sh
git clone https://github.com/gripoio/gripo-docker-deployment.git
```

After cloning the repository, navigate to the project directory by executing the following command.

```sh
cd gripo-docker-deployment
```

Execute the following command.

```powershell
cd gripo-mac/
```

Than execute the following command.

```powershell
sudo docker-compose up -d
```
**Note**: To confirm that clone is compose accurately , execute a command.



```
docker ps
```
![licence](/licence/licence2.png)

If the containers listed in the image do not appear after executing the docker ps command, you should re-run the following command to ensure the services are up and running:

```
docker-compose up -d
```

This ensures all services defined in the Docker Compose file are started properly.
