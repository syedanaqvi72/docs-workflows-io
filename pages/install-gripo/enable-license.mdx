# Enable Gripo license

Gripo requires a license to provision Kubernetes clusters. There are multiple ways to enable a Gripo license.

- Request a license from the [Gripo website](https://www.gripo.io/request-free-license).
- You will also be presented with the screen below when you access Gripo dashboard for the first time, which can direct you to the license request form.

![overview](/install-gripo/enable-license/l1.png)

Fill in the form below to request a license. You will be sent a license key via email once your request is accepted.

![overview](/install-gripo/enable-license/l4.png)

Press add button and enter the license key received on your email.

![overview](/install-gripo/enable-license/l2.png)

You will be presented with below screed which gives shows the license detail. Refresh the page to show the license details .

![overview](/install-gripo/enable-license/l3.png)
