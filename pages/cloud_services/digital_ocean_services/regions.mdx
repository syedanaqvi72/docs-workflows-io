#### <b> Digital Ocean Regions </b>

The region plays a vital role in cluster creation, impacting performance, compliance, reliability, and cost. Gripoâ€™s integration with DigitalOcean and other cloud providers allows users to strategically choose regions for the best possible results.

* To access Google Services, navigate to <b>Settings</b>, click on Cloud Services.
<br />
![DG-k8s](/cloud_services/cloud_services.png)<br />

* Select <b>Digital Ocean.</b>
<br />
![dg-region](/cloud_services/digital_ocean/digital-cloud.png)<br />

* Select <b>Regions.</b>
<br />
![dg-region](/cloud_services/digital_ocean/digial-operation.png) <br />
* Click on <b>Add Regions</b> button.

<br />![dg-Region](/cloud_services/digital_ocean/regions/region0.png) <br />

* To add Regions, the user must first fetch the latest updates. Click the Sync button to retrieve the updated Regions.
<br />
<br />![dg-Region](/cloud_services/digital_ocean/regions/region2.png) <br />

* A confirmation sync pop-up window will appear.
* Select the Secret.
<br />![dg-Region](/cloud_services/digital_ocean/regions/region3.png) <br />


* After syncing, a window opens with available regions. Simply select a region and click the "Add Region" button in the top-right corner to create and activate it instantly
