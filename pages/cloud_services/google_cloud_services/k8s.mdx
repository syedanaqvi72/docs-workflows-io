#### Google's K8s Service into the Gripo Platform

Unlocking the potential of Google's Kubernetes (K8s) service with the Gripo platform is simple and efficient. Follow these easy steps to ensure a smooth integration process:

<b>Select K8s Configuration:</b>
* To activate Google Kubernetes (K8s) within the Gripo platform, click on Cloud Services.
<br />
![google-k8s](/cloud_services/cloud_services.png)<br />

* Select <b>Google.</b>
<br />
![linode-k8s](/cloud_services/google/googlecloud.png)<br />

* Select <b>K8s.</b>
<br />
![Google Region](/cloud_services/google/region/googleregion1.png) <br />
<b> Add K8s:</b>

* Click on the Add Kubernetes (K8s) button.<br />
![linode-k8s](/cloud_services/google/k8s/google-k8s2.png)<br />
* A pop-up window will appear. In this window, write the desired Kubernetes version and click the Add Version button.
<br />
![linode-k8s](/cloud_services/google/k8s/google-k8s3.png)<br />
<br />
* A confirmation window for adding Kubernetes (K8s) will appear.
<br />
![linode-k8s](/cloud_services/google/k8s/google-k8s4.png)<br />
<br />
* The added Kubernetes (K8s) will be displayed in the list under the Google K8s section.
<br />
![linode-k8s](/cloud_services/google/k8s/google-k8s5.png)<br />
<b> Delete K8s:</b>
* To delete a Kubernetes (K8s) entry, first select the K8s instance you wish to remove.Once selected, click the Delete button. 

<br />
![linode-k8s](/cloud_services/google/k8s/google-k8s5.png)<br />
* A confirmation window will appear; confirm the action, and the selected K8s will be removed from the list.
<br />![linode-k8s](/cloud_services/google/k8s/google-k8s6.png)<br />
