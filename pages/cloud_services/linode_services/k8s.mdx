#### Linode's K8s Service into the Gripo Platform

Unlocking the potential of Linode's Kubernetes (K8s) service with the Gripo platform is simple and efficient. Follow these easy steps to ensure a smooth integration process:

<b>Select K8s Configuration:</b>
* To activate Linode Kubernetes (K8s) within the Gripo platform, click on Cloud Services.
<br />
![linode-k8s](/cloud_services/cloud_services.png)<br />

* Select <b>Linode.</b>
<br />
<br />
![linode-k8s](/cloud_services/linode/linodecloud.png)<br />
* Select <b>K8s.</b>
<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s1.png)<br />
<b> Add K8s:</b>
* Click on the Add Kubernetes (K8s) button.<br />

![linode-k8s](/cloud_services/linode/k8s/linodek8s2.png)<br />
* To add K8s, the user must first fetch the latest updates.
* Click the Sync button to retrieve the updated k8s.
<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s3.png)<br />
* A pop-up window will appear. Select your secret .
<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s4.png)<br />
* Select the desired Kubernetes version and click the Add Version button.

<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s5.png)<br />
<br />

* A confirmation window for adding Kubernetes (K8s) will appear.
<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s6.png)<br />
* The added Kubernetes (K8s) will be displayed in the list under the Linode K8s section.
<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s7.png)<br />
<b> Delete K8s:</b>
* To delete a Kubernetes (K8s) entry, first select the K8s instance you wish to remove.Once selected, click the Delete button. 

<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s7.png)<br />
* A confirmation window will appear; confirm the action, and the selected K8s will be removed from the list.
<br />
![linode-k8s](/cloud_services/linode/k8s/linodek8s8.png)<br />
