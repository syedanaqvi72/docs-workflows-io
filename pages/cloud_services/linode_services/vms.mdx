#### Linode Virtual Machines

* To activate Linode Kubernetes (K8s) within the Gripo platform, click on Cloud Services.
<br />
![Azure-k8s](/cloud_services/cloud_services.png)<br />

* Select <b>Linode.</b>
<br />
![linode-vmss](/cloud_services/linode/linodecloud.png)<br />

* Select <b>VMs</b>
<br />
![linode-vms](/cloud_services/linode/k8s/linodek8s1.png)<br />
* Click on Add Vms
<br />
![linode-vms](/cloud_services/linode/vms/linvms2.png)<br />
* To add virtual machines (VMs), the user must first fetch the latest updates. Click the Sync button to retrieve the updated VMs.

* A pop-up window will appear. Select the Secret.
* Click on Sync Button.
<br />
![linode-vms](/cloud_services/linode/vms/linvms3.png)<br />

* After synchronization, users can add a VM by selecting one from the list and clicking the <b>"Add Virtual Machine"</b> button.

<br />
![linode-vms](/cloud_services/linode/vms/linvms4.png)<br />

* With Gripo, users can easily remove mistakenly selected VMs by choosing them from the list and clicking the <b>"Delete"</b> button to start the deletion process.

<br />
![linode-vms](/cloud_services/linode/vms/linvms5.png)<br />
