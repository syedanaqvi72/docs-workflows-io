#### Regions in Linode

The region is crucial for cluster creation, affecting performance, compliance, resilience, and cost. Gripoâ€™s integration with Linode and other cloud providers enables strategic region selection for optimized outcomes.

* To access Linode Services, navigate to <b>Settings</b>, click on Cloud Services.
<br />
![Linode-region](/cloud_services/cloud_services.png)<br />

* Select <b>Linode.</b>
<br />
![linode-region](/cloud_services/linode/linodecloud.png)<br />

* Select <b>Regions.</b>
<br />
![linode-Region](/cloud_services/linode/vms/linvms1.png) <br />

* Click on <b>Add Regions</b> button.

<br />![linode-Region](/cloud_services/linode/region/linreg1.png) <br />

* To add Regions, the user must first fetch the latest updates.
* Click the Sync button to retrieve the updated Regions.
<br />
![linode-Region](/cloud_services/linode/region/linreg2.png) <br />
<br  />
* A sync pop-up window will appear.
* Select the Secret And than click on Select Secret.
<br />
![linode-Region](/cloud_services/linode/region/linreg3.png) <br />
<br  />
* After synchronization, a window will display a list of available regions.
* Select your preferred region and click the <b>Add Region</b> button to create add region immediately.
<br />
![linode-Region](/cloud_services/linode/region/linreg4.png)
<br  />
* A confirmation add region pop-up window will appear.

<br />
<br  />

<b> Delete Regions: </b>
* Gripo allows users to remove mistakenly added Regions by selecting the Region from the list .
* Click the Delete button, which will permanently delete the Region from your list on the Gripo platform.
<br />
![linode-Region](/cloud_services/linode/region/linreg5.png)
<br />
* A confirmation delete pop-up will appear. Once confirmed, the selected region will be removed and no longer appear in the list.
<br />
![linode-Region](/cloud_services/linode/region/linreg6.png)
