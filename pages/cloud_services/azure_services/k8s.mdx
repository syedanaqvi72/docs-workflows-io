#### Azure's K8s Service into the Gripo Platform

Unlocking the potential of Azure's Kubernetes (K8s) service with the Gripo platform is simple and efficient. Follow these easy steps to ensure a smooth integration process:

<b>Select K8s Configuration:</b>
* To activate Azure Kubernetes (K8s) within the Gripo platform, click on Cloud Services.
<br />
![Azure-k8s](/cloud_services/cloud_services.png)<br />

* Select <b>Azure.</b>
<br />
![Azure-k8s](/cloud_services/azure/azurecloud.png)<br />
* Select <b>K8s.</b>
<br />
![Azure-k8s](/cloud_services/azure/k8s/azurek8s1.png)<br />
<b> Add K8s:</b>

* Click on the Add Kubernetes (K8s) button.<br />
![Azure-k8s](/cloud_services/azure/k8s/azurek8s2.png)<br />
* A pop-up window will appear. In this window, write the desired Kubernetes version and click the Add Version button.
<br />
![Azure-k8s](/cloud_services/azure/k8s/azurek8s3.png)
<br />
* A confirmation window for adding Kubernetes (K8s) will appear.
<br />
![Azure-k8s](/cloud_services/azure/k8s/azurek8s4.png)
<br />
* The added Kubernetes (K8s) will be displayed in the list under the Azure K8s section.
<br />
![Azure-k8s](/cloud_services/azure/k8s/azurek8s5.png)<br />
<b> Delete K8s:</b>
* To delete a Kubernetes (K8s) entry, first select the K8s instance you wish to remove.Once selected, click the Delete button. 

<br />
![Azure-k8s](/cloud_services/azure/k8s/azurek8s6.png)<br />
* A confirmation window will appear; confirm the action, and the selected K8s will be removed from the list.
<br />
![Azure-k8s](/cloud_services/azure/k8s/azurek8s7.png)<br />
