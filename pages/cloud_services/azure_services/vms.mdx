#### Vms of Azure

* To activate Azure Kubernetes (K8s) within the Gripo platform, click on Cloud Services.
<br />
![Azure-k8s](/cloud_services/cloud_services.png)<br />

* Select <b>Azure.</b>
<br />
![Azure-k8s](/cloud_services/azure/azurecloud.png)<br />

* Select <b>VMs</b>
<br />
![Azure-vms](/cloud_services/azure/vms/azurevms0.png)<br />
* To add virtual machines (VMs), the user must first fetch the latest updates. Click the Sync button to retrieve the updated VMs.
<br />
![Azure-vms](/cloud_services/azure/vms/azurevms2.png)<br />
* A pop-up window will appear. Select the Secret and select the Region.

<br  /> 
![VM-Azure](/cloud_services/azure/vms/vms-sync.png)<br />

* Once the synchronization process is complete, users can proceed to create a virtual machine (VM) by selecting their desired VM from the available list and clicking the "Add Virtual Machine" button. 
<br />
![VM-Azure](/cloud_services/azure/vms/selected-vms.png)<br />
<b> Delete Vms: </b>
* Gripo allows users to remove mistakenly selected Virtual Machines (VMs) by selecting the VM from the list and clicking the Select and Delete button.
<br />
![VM-Azure](/cloud_services/azure/vms/selected-list.png)

* To remove a selected VM, click the Delete button, which will permanently delete the VM from your list on the Gripo platform.
<br />
![VM-Azure](/cloud_services/azure/vms/delete-vms.png)
