#### Regions in Azure

The region is crucial for cluster creation, affecting performance, compliance, resilience, and cost. Gripoâ€™s integration with Azure and other cloud providers enables strategic region selection for optimized outcomes.

* To access Azure Services, navigate to <b>Settings</b>, click on Cloud Services.
<br />
![Azure-k8s](/cloud_services/cloud_services.png)<br />

* Select <b>Azure.</b>
<br />
![Azure-k8s](/cloud_services/azure/azurecloud.png)<br />

* Select <b>Region.</b>
<br />
![Azure Region](/cloud_services/azure/regions/azureregion1.png) <br />
* Click on <b>Add Regions</b> button.
<br />
![Azure Region](/cloud_services/azure/regions/azureregion2.png) <br />
* To add Regions, the user must first fetch the latest updates. Click the Sync button to retrieve the updated Regions.
<br />
![VM-Azure](/cloud_services/azure/regions/azure-sync.png)
<br />
* A pop-up window will appear. Select your secret from the dropdown menu.
![Azure-Region](/cloud_services/azure/regions/azureregion3.png)<br  />
* Click on <b> Sync </b> button.
<br />
![Azure-Region](/cloud_services/azure/regions/azureregion4.png)<br  />
* A confirmation sync pop-up window will appear.
<br />
![Azure-Region](/cloud_services/azure/regions/azureregion5.png)<br  />
* After synchronization, a window will display a list of available regions.
* Select your preferred region and click the <b>Add Region</b> button to create add region immediately.
<br />
![Azure-Region](/cloud_services/azure/regions/azureregion6.png)<br  />
* A confirmation add region pop-up window will appear.

<br />
![Azure-Region](/cloud_services/azure/regions/azureregion7.png)<br  />

<b> Delete Regions: </b>
* Gripo allows users to remove mistakenly added Regions by selecting the Region from the list .
<br />
![VM-Azure](/cloud_services/azure/vms/selected-list.png)

* Click the Delete button, which will permanently delete the Region from your list on the Gripo platform.
<br />
![Azure-Region](/cloud_services/azure/regions/azureregion8.png)<br  />
* A confirmation delete pop-up will appear. Once confirmed, the selected region will be removed and no longer appear in the list.
<br />
![Azure-Region](/cloud_services/azure/regions/azureregion9.png)<br  />
