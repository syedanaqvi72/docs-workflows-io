## Linode's Secret

To integrate your GRiPO application with Linode services like LKE, follow these steps

- [Create Secret on Gripo](creating-secret#create-secret-on-gripo)
- [Create Token from Linode Portal](creating-secret#create-token-from-linode-portal)

#### Create Secret On Gripo
- To create a secret, navigate to the <b> Administration </b>section and select<b> Settings.</b>

![azure](/secret/azure/azuresecret.png)<br />

- Click on <b>Secret</b><br />

![azure](/secret/azure/azuresecret0.png)<br />


- To initiate the secret creation process, click on the <b>Create</b> button.
  ![azure](/secret/azure/azuresecret01.png)<br />
- A pop-up window will appear displaying various cloud providers when Proceed by selecting <b> Linode.</b>

![azure](/secret/azure/azuresecret19.png) <br />

- Specify <b>'Linode'</b> as the service provider.
- Enter a <b>designated name</b> for your account.
- Provide the <b>'Secret.'</b>.

![linode](/secret/linode/linode-secret5a.png)<br />
- To fill in the required fields, Personal Access Token must first be created in the Linode Portal.


#### Create Token from Linode Portal

- Visit https://login.linode.com/.
- Click <b>"Sign In" </b>after providing username and password.

![linode](/secret/linode/linode-secret1.jpg)

- Upon successfully signing in to the Linode interface, proceed to access the API Token functionality by clicking on<b> API Tokens</b>

![linode](/secret/linode/linode-secret2.png)

- Within the API Tokens section, locate and select <b>"Create a Personal Access Token"</b> to initiate the token creation process.

![linode](/secret/linode/linode-secret3.png)

Complete the form by providing the necessary information:

- <b>Label: </b>Assign a label to the token for identification purposes.
- <b>Expiry Duration:</b> Set the expiration duration for the token.
- Ensure that you select <b>"Read and Write"</b> permissions for all relevant options.
- Click <b>"Create Token"</b> to generate the access token

![linode](/secret/linode/linode-secret4.png)

- A pop-up window will appear, displaying the generated personal access token.

![linode](/secret/linode/linode-secret5.png)<br />
- Paste the copied <b>Token</b> from the Linode portal into the <b>Secret </b>field.
- Select the <b>"Submit"</b> button to confirm and complete the process.

![linode](/secret/linode/linode-secret6.png)

- Your recently created secret is now visible in the list of secrets for your reference.
