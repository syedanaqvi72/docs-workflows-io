## Google Secret Documentation

You will need to create a Google secret to communicate with Google Kubernetes services. Sensitive data, including passwords, authentication keys, and certificates from the Google provider, can be safely managed and stored using GRiPO. After creating the secret on the Google platform, add it to GRiPO.

### Getting Started

To get started with Google Secret, you have to create  [Google secret](/secret/google/creating-secret).
