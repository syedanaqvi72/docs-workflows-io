### Add Repository <br />:

<b> To add a Helm application repository in Gripo, follow these steps:</b><br />
* Click <b>"Settings"</b>.
<br />
![repo](/app-deploy/add-repo/add-repo1.jpeg)<br  />
* Click <b>"Manage Repositories".</b>
<br />
![repo](/app-deploy/add-repo/add-repo2.jpeg)<br  />
* Click <b>"Add Helm Package Repo"</b> 
 <br />
![repo](/app-deploy/add-repo/add-repo3.jpeg)<br  />
* A window will appear with fields for Package Name and URL.
<br />
![repo](/app-deploy/add-repo/add-repo4.jpeg)<br  /> 
* Enter the package name and the repository URL you wish to add.
* Click the <b>"Add Repo"</b>
<br />
![repo](/app-deploy/add-repo/add-repo5.png)<br  />
* Once added, the repository will be visible in the repository list.
<br />
![repo](/app-deploy/add-repo/add-repo6.jpeg)<br  />
