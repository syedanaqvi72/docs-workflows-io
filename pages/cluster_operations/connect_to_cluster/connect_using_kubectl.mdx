- Execute the command line: <b>"kubectl get nodes "</b>to access detailed information about the nodes in your connected cluster

![kubectl](/cluster_operation/connect_to_cluster/kubectl/kubectl3.png)

- For an overview of active services in the cluster, use <b>"kubectl get service"</b> for essential details like names, types, IPs, and ports.

![kubectl](/cluster_operation/connect_to_cluster/kubectl/kubectl4.png)

- To obtain a comprehensive overview of historical events within the Kubernetes cluster, utilize the professional command: <b>"kubectl get event" </b>
